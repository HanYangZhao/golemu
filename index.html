Gologir

A mock LLRP-based RFID reader for RFID information system testing

About

Gologir is a sample implementation of an RFID emulator for practitioners/researchers of RFID-based automatic identification systems. The emulator significantly mitigates the RFID information system testing process and encourages more in performance evaluation by completely eliminating the physical tags placement and reading with physical interrogator.

Features

  • Generate "Multicode" (ISO, GS 1 and even proprietary) RFID tag events
  • Dynamic tag population control simulating moving tags through web interfaces (Websocket/REST)
  • Communicate with a middleware via Low Level Reader Protocol (LLRP)
